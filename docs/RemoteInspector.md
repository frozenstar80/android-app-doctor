# AppDoctor Remote Inspector

`appdoctor-remote` is not part of the current repository and is not shipped in `v1.0.0`.

## Current status

- No runtime remote transport is active.
- No automatic off-device streaming or control path exists in this release.
- Extension capability enums reserve remote command semantics for future expansion.

## Planned direction

- Opt-in remote transport module with explicit host configuration
- Strict redaction controls and transport security requirements
- Non-breaking additive integration with timeline/session/diagnostics surfaces
